## 项目核心定位

**一句话**：让 AI Agent 成为去中心化商业协议的原生参与者，
实现自动卖货、自动买货，所有人几乎零成本参与。

**核心差异**：
- 不是"新平台要用户迁移"，而是"给现有 Agent 增加商业能力"
- 每个状态转移都需要责任方自举证，协议自动判责
- Agent Skill 市场解决冷启动，商家零额外工作量接入新渠道
- 所有角色仅需手机参与

**目标用户（冷启动顺序）**：
1. 现有 AI Agent 重度用户（技术先行者）
2. 已有店铺的电商卖家（零成本新渠道）
3. 习惯让 Agent 购物的普通买家

## 技术栈决策

| 方向 | 选择 | 原因 |
|------|------|------|
| 运行环境 | Node.js | 已安装，MCP SDK 是 JS |
| 数据库 | SQLite → PostgreSQL | 简单起步，后期迁移 |
| Agent 接口 | MCP (Model Context Protocol) | Anthropic 标准，Claude 原生支持 |
| 前端 | PWA（渐进式网页应用） | 手机浏览器直接用，无需下载 |
| 存储（Phase 0） | 本地文件系统 | 先跑通逻辑 |
| 存储（Phase 2） | IPFS | 去中心化 |
| 链（Phase 2） | 待定（Base/Optimism 候选） | 低 gas，EVM 兼容 |
| 语言 | TypeScript | 类型安全，MCP SDK 原生支持 |
